# MineOS Consent Management Platform — Google Tag Manager Template

This repository contains the official Google Tag Manager (GTM) custom template for the MineOS Consent Management Platform (CMP). The template lets you deploy and configure the MineOS CMP on your website directly from your GTM container, without editing site code.

All settings and configurations are managed through the [MineOS portal](https://mineos.ai/platforms/consent-management), which allows you to customize the CMP's behavior and appearance according to your needs.

# Importing the template into your GTM container

1. Download the template.tpl file from this repo.
2. Open your GTM container and navigate to the "Templates" section.
3. Click on "New" and then select "Import" to upload the template.tpl file.
4. Once imported, you can create a new tag using the MineOS CMP template and configure it according to your requirements.
